# PAS 🏛️💻

Este repositorio contiene una colección de algoritmos y soluciones desarrolladas en **Pascal**, el lenguaje pilar de la programación estructurada y la enseñanza de la computación.

## 🚀 Descripción

`PAS` es un proyecto de fundamentos de ingeniería de software que compila diversas implementaciones lógicas utilizando la sintaxis clara y tipado fuerte de Pascal. El repositorio sirve como un registro histórico y técnico de la evolución del autor en el diseño de algoritmos, cubriendo desde estructuras de datos básicas hasta lógica matemática pura.

Representa el rigor académico y la disciplina de programación necesarios para construir sistemas robustos y eficientes desde la base.

## 🛠️ Especificaciones Técnicas

- **Lenguaje:** Pascal (Turbo Pascal / Free Pascal).
- **Paradigma:** Programación Estructurada y Modular.
- **Funcionalidades Clave:**
  - **Algoritmia Clásica:** Implementación de búsquedas, ordenamientos y manejo de arreglos.
  - **Estructuras de Datos:** Gestión de registros (Records), punteros y listas enlazadas.
  - **Matemática Aplicada:** Resolución de ecuaciones y cálculos trigonométricos de baja abstracción.
  - **Optimización de Memoria:** Uso eficiente de recursos en entornos de computación de recursos limitados.

## 📂 Estructura del Repositorio

Tras la reestructuración profesional:
- `/src`: Archivos de código fuente (.pas) organizados por categorías.
- `/bin`: (Opcional) Ejecutables compilados para entornos compatibles.
- `/docs`: Notas sobre la lógica de Pascal y manuales de sintaxis aplicados.
